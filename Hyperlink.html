Link / Hyperlink : Elemen Untuk Menghubungkan Halaman Web

Link / Hyperlink adalah elemen HTML yang berfungsi menghubungkan suatu halaman web ke halaman web yang lain. Elemen ini bisa diklik dan nanti akan membuka halaman lain sesuai alamat URL yang diberikan. link juga punya fungsi lain seperti scroll top, download file, menjalankan fungsi javascript, dll.

Klik Untuk Melihat Contohnya :

Jenis-Jenis Link

Link Statis / Internal

Link Dinamis / External

Selain untuk berpindah halaman, link juga digunakan untuk tooltip dan anchor dari sebuah link.

JavaScript juga dapat dijalankan dengan link, namun cukup jarang dipakai.
